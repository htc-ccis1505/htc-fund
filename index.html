--- layout: page title: "Welcome to Fundamentals of Programming!" ---

This course is an introduction to computer programming. In this course, you'll learn to think like a programmer. The focus of the course is on programming logic and structures - not learning a particular programming language. We'll begin with a graphical language called Snap! and then transition into working with Python at the end of the course.

Requirements

There are a few things you need in order to be successful in this course:

  1. A GitHub account. Our course materials are hosted by GitHub, some of which may be private and require access to the CCIS 1505 organization to view. Set yourself up with an account - I encourage you to use your actual name, then follow me (htcMosman), and email me with both your real name and account name. I will then add you to the course organization so that you can access our Gitter chat room and any private data.
  2. The Google Chrome web browser. It is recommended for working with Snap!.
  3. A willingness to learn. This course can be challenging, but fun. Persistence and a positive outlook go a long way.
  4. Teamwork. We will regularly work in pairs in the classroom labs. Make friends, have fun, and support each other. Don't try to go it alone.
